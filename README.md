# Meal-Order-App
This is a JavaScript application that interacts with [TheMealDB](https://www.themealdb.com/api.php) API to take and complete orders. It utilizes API methods to filter meals by ingredients and keeps "memory" of orders using sessionStorage.

---

## Features
- Simple clean design with a professional look
- Easy navigation (single page)
- Cards for structured content and sectioning
- Works seamlessly on desktop and mobile

---

## Technologies used:
- HTML5
- CSS3
- Bootstrap 5
- Github Pages for hosting
